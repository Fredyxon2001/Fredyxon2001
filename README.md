### 🚀 En qué trabajo

Diseño y opero un **sistema de gestión de cobranzas y bonos** de punta a punta: motor de reglas de bono por niveles, subasta interna de cuentas, cierre semanal auditable con firma del director, tablero en tiempo real y reportería. Todo corriendo en producción, con cierres automatizados cada semana y conciliación contra el sistema legacy.

- 🧮 **Motores de bono deterministas** — reglas versionadas, excepciones firmadas y cálculo idéntico entre pantalla y reporte.
- 🔄 **Pipelines de datos** — snapshots por hora desde MySQL legacy + BigQuery hacia Postgres, con candados de cierre e idempotencia.
- 🗺️ **Visualización** — rankings jerárquicos, mapas de calor por estado y dashboards en vivo.
- ⚙️ **Automatización** — crons en GitHub Actions para pagos, radares de metas y auditorías pre-cierre.
